# 项目简介 java777

本项目是基于Springboot和Vue的作业管理系统，采用前后端分离的技术架构。系统支持用户和管理员两种角色的登录，旨在为用户提供一个便捷、高效的管理平台。

## 技术栈

- 后端：Java、Springboot、MyBatis、Maven
- 前端：Vue
- 数据库：MySQL

## 运行环境

- JDK 1.8
- Maven
- MySQL 5.7
- Node.js

## 功能模块

系统主要包括以下功能模块：

1. 用户登录与注册
2. 作业提交与批改
3. 管理员后台管理
   - 用户管理
   - 作业管理
   - 系统设置

## 系统角色

- 用户：可以提交作业、查看作业成绩和修改个人资料
- 管理员：负责管理用户、作业和系统设置

## 部署步骤

1. 准备运行环境：安装JDK 1.8、Maven、MySQL 5.7和Node.js
2. 创建数据库：导入项目提供的数据库文件
3. 编译后端代码：使用Maven命令编译后端代码
4. 部署前端代码：在项目前端目录下运行`npm install`和`npm run build`命令，将构建的文件放入后端项目的指定目录下
5. 启动后端服务：运行后端项目的主程序
